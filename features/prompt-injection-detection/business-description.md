# Prompt Injection Detection — Business-Oriented Description

## Executive Summary

**Prompt Injection Detection** protects AI agents from manipulation attacks where hidden instructions are inserted into inputs to make agents behave maliciously.

## The Problem

| Attack Type | Impact |
|-------------|--------|
| Hidden instructions | Agent ignores safety rules |
| Context switching | Agent adopts attacker persona |
| Encoded payloads | Obfuscated malicious commands |
| Role-playing | Agent "acts as" attacker |
