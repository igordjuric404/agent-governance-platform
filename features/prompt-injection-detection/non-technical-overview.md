# Prompt Injection Detection — Non-Technical Overview

## What Is Prompt Injection Detection?

**Prompt injection** is a technique where attackers try to manipulate AI agents by inserting hidden instructions into their inputs. **Prompt Injection Detection** identifies and blocks these attacks.

## Why It Matters

Without detection:
- **Hidden commands** — Attackers hide instructions in text
- **Context override** — Agent ignores its safety instructions
- **Data theft** — Agent could be tricked into revealing secrets
- **Action manipulation** — Agent could be made to take malicious actions

## How It Works

Detection analyzes inputs for injection patterns:
1. **Hidden instructions** — Text that tries to override behavior
2. **Context switching** — Attempts to change conversation context
3. **Payload encoding** — Obfuscated commands
4. **Role-playing attacks** — Attempts to make agent "act as" something else

## Key Benefits

- Protection against prompt injection attacks
- Detection of hidden malicious content
- 12-vector analysis coverage
- Automatic blocking of suspicious inputs

## The Bottom Line

Prompt Injection Detection protects your AI agents from manipulation attempts hidden in their inputs — ensuring agents do what you intended, not what attackers want.
