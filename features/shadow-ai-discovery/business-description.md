# Shadow AI Discovery — Business-Oriented Description

## Executive Summary

**Shadow AI Discovery** finds AI agents running in your organization that you don't know about — the first step to governing them properly.

## The Problem

| Risk | Impact |
|------|--------|
| Unknown agents | Unmanaged risk |
| Unapproved AI | Compliance violations |
| Security blind spots | Potential breaches |
| Duplicate agents | Wasted resources |

## Value Proposition

| Without Discovery | With Shadow AI Discovery |
|------------------|------------------------|
| Unknown inventory | Complete visibility |
| Unmanaged risk | All agents tracked |
| Compliance gaps | Automated discovery |
| Security blind spots | Full coverage |

## Key Features

- **Multi-Scanner Architecture** — Process, config, repository, network
- **Risk Scoring** — Discovered agents evaluated for risk
- **Inventory Management** — Complete agent registry
- **Automated Sync** — Inventory stays current

## Use Cases

1. **Security Audits** — Find all AI agents in organization
2. **Compliance** — Ensure all agents are approved
3. **Cost Optimization** — Identify duplicate/unused agents
4. **M&A Due Diligence** — Discover agents in acquisitions
